# Hami MiniMarket - E-commerce Website

## Project Overview
A modern, responsive e-commerce website for fresh groceries and produce, featuring a complete shopping cart system with persistent storage.

## Features

### Core Functionality
- **Product Catalog** - 16 fresh grocery items with images and prices
- **Shopping Cart** - Add, remove, and update product quantities
- **Persistent Storage** - Cart data saved using localStorage
- **Order Management** - Complete checkout process with order confirmation
- **Responsive Design** - Mobile-first approach works on all devices

### User Experience
- **Interactive Cart** - Slide-out sidebar for cart management
- **Real-time Calculations** - Automatic tax and discount calculations
- **Smooth Animations** - CSS transitions and hover effects
- **Toast Notifications** - User feedback for cart actions
- **Professional UI** - Clean, modern interface with navy blue theme

### Technical Features
- **Modular JavaScript** - Organized code structure
- **CSS Grid & Flexbox** - Modern layout techniques
- **LocalStorage API** - Data persistence across sessions
- **Font Awesome Icons** - Professional iconography
- **Unsplash Images** - High-quality product photos
